## 💻 Iniciando o projeto

**Clone o projeto e acesse a pasta**

```bash
$ git clone https://github.com/msalves008/ioasys-desafio-books-frontend && cd ioasys-desafio-books-frontend
```

**Siga os passos a baixo**

```bash
# Instalando as dependências
$ yarn install


# Iniciando o projeto
$ yarn dev
```
